Template Parameters
IThe type used to store an IBEA indicator
SThe type used to store an IBEA selection operator
MThe type used to store and IBEA mutation operator
CThe type used to store and IBEA crossover operator
Parameters
maxevalThe maximum number of evaluations performed by the algorithms (stopping criterion)
population_max_sizeThe maximum population size
max_generationsThe maximum number of generations
scaling_factorThe scaling factor
indicatorThe indicator to be used by the IBEA indicator operator
crossover_methodThe crossover method considered by the IBEA mutation operator
mutation_methodThe mutation method considered by the IBEA mutation operator
selection_methodThe selection method considered by the IBEA selection operator
adaptiveboolean indicative of version of IBEA to be used. If true use adaptive version of (A-IBEA) else use (B-IBEA)
